# 🏠 PropertyHub – Rental Management System

PropertyHub is a full-stack **Rental Management Platform** designed to digitize and streamline property operations between **property owners, tenants, and maintenance teams**. It provides an end-to-end workflow covering onboarding, property allocation, complaint handling, service management, billing, and agreements.

---

## 🚀 Overview

PropertyHub acts as a centralized system where:

- Owners can manage properties, tenants, and earnings
- Tenants can register, raise complaints, and access documents
- Technicians can be assigned for maintenance tasks
- Automated invoices and agreements are generated dynamically

The system is designed to simulate a real-world property management ecosystem with role-based access and automated backend workflows.
