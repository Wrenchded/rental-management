PropertyHub – Rental Management System

A full-stack Rental Management System built using FastAPI, SQLite, and Jinja2 templates, designed to streamline property operations between owners, tenants, and maintenance teams.

This project includes role-based dashboards, automated workflows, maintenance ticketing, invoice generation, and a mobile-friendly web interface.

📌 Features
🧑‍💼 Owner Module
Owner registration & authentication flow
Property configuration (PG / Flat / Office / Chawl / Commercial)
View dashboard with:
Occupancy tracking
Revenue estimation
Active maintenance tickets
Assign technicians to complaints
Resolve tickets and generate invoices

🏠 Tenant Module
Tenant onboarding system
Dynamic property allocation
Raise maintenance complaints
View invoices & agreements
Access tenant dashboard

🛠️ Maintenance System
Raise maintenance tickets
Category-based complaint tracking
Technician assignment system
Service logs with cost calculation
Auto-generated invoice system

Agreement System
Auto-generated rental agreements
Owner + tenant contract details
Payment breakdown (commission + earnings)
